# Changelog

All notable changes to Perfect Playlist are documented here.

## 0.1.0 - 2026-07-18

Initial public release.

- Build new public playlists or fill owned empty public/private targets from exact Sources.
- Append exact tracks safely with Add and post-write verification.
- Compare Sources with count-first, first-difference Verify diagnostics.
- Export canonical YAML, JSON, text, or Spotify links without overwriting files.
- Search and Inspect Spotify tracks without implicit selection or playlist writes.
- Support a deterministic importable Python API with typed results and failures.
